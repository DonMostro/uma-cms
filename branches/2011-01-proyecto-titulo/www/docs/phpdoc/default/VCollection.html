Class VCollection

Description
Description | Vars (details) | Methods (details)

Base view class to display the collection data types. The inherited classes implements decorate_list() and decorate_item() methods. The main idea for these methods (and the View layer in general) is to create tags for the templates and insert them into the selected templates. The methods return the generated content.

Since version 3.0, decorate_list() and decorate_item() can be omitted. In that case, filter() is used to decorate items that must be processed rather than fetching to the template straight from the model.
